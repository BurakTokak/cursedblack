Cursed Black is an experimental game that focuses on developing your foresight. You will allow to make 3 moves before you make a line or column blocks all same color. If you can't, a black block capture one of your blocks.

By the way, you will challenge with the random block color changings. After every move of yours, the game will randomly change a block which is not a black block.

The Game will end when you don't have the space to tile any lines or columns with the same color. (12+ black blocks)
